Features

Everything you need,
nothing you don't.

🛑
Hard budget enforcement
Set daily and monthly limits. When you hit the cap, ClawCost returns an HTTP 429 before the API call is even made — not after the bill lands.
📊
Per-agent, per-model tracking
See exactly which agent or session is spending what, broken down by model. Identify your most expensive calls in seconds.
⚡
Zero latency streaming
ClawCost parses SSE events as they stream through — it never buffers the full response. Your agents see no speed difference.
🔌
One line to activate
No OpenClaw plugins to install. No config changes. Just point OpenClaw's base URL at ClawCost and you're done.
# ~/.openclaw/.env
$ npm run dev
ClawCost proxy → http://localhost:4100/v1
Dashboard → http://localhost:4100
 
# Add to ~/.openclaw/.env and restart
ANTHROPIC_BASE_URL=http://localhost:4100/v1
🔒
100% local, private by default
Usage data is stored in a local SQLite database at ~/.clawcost/usage.db. Nothing leaves your machine.
🌐
Anthropic + OpenAI
Works with all major providers OpenClaw supports. Accurate pricing for Claude, GPT-4o, o3, and Gemini models built in.
How it works

Up and running
in under 5 minutes.

Step 01
Install and start the proxy
Clone the repo, run npm install, and start ClawCost. It binds to localhost:4100.
$ npm run dev
Step 02
Point OpenClaw at ClawCost
Add one line to your OpenClaw environment file. That's the only change needed — no plugins, no code modifications.
ANTHROPIC_BASE_URL=http://localhost:4100/v1
Step 03
Set your budget and go
Open the dashboard, set your daily and monthly limits, and forget about it. ClawCost handles the rest.
Open → http://localhost:4100
